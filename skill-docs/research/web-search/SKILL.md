# Web Search
## When to Use
- **DO search for:** News, recent events (last 6 months), stock prices, product releases.
- **DO NOT search for:** Historical facts, scientific principles, math formulas.
## Tools
- `search_web(query)`: General search.
- `search_webpage(query, webpage)`: Search within a specific URL.
